# Log Monitoring System

A Flask-based web application for monitoring and analyzing log files.

## Features

- Upload log files
- Parse logs automatically
- Store logs in SQLite database
- Dashboard with statistics
- Pie chart visualization
- Search logs
- Pagination
- CSV export
- Deployed on Render

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
